# Port Scan Tool

## Desenvolvedor:
- José Rafael Martins Fernandes

## Sobre o projeto:
-  O Port Scan Tool é uma command-line interface que permite ao usuário fazer as seguintes operações:
    - Scanear uma rede local e retornar os hosts ativos e suas portas abertas 
    - Scanear uma rede local e retornar os hosts ativos e suas portas abertas, a partir de um range de portas definido pelo usuário
    - Scanear uma rede local e retornar os hosts ativos e uma relação das portas mais conhecidas abertas

## Dependencias:
* Python 3.8+ (https://www.python.org/downloads)
* nmap (https://nmap.org/download)
